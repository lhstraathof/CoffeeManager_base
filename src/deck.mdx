# Resources

##### React docs:
The official docs are verry well written and a recommended
[React JS](https://reactjs.org/docs/getting-started.html)

##### Create app:
Tool by facebook to create your react app without the need to think about tooling. Saves a lot of headache
[Create React App](https://facebook.github.io/create-react-app/)

##### Routing
Populair library to setup routing in React
[React Router](https://reacttraining.com/react-router/web/guides/quick-start)

##### Unit-Testing
Works verry well with React JS as it is build by the same team. But is a great stand-alone tool
[Jest.io](https://jestjs.io/)

##### Material UI Components:
The UI components used in this workshop
[Material UI](https://material-ui.com)

##### Component overview:
Great tool for working component based. Allows you to build a component in a isollated environment
[Storybook](https://storybook.js.org/)

##### Deploy app:
Tool to deploy your fresh app instantly
[Zeit Now](https://zeit.co/now)

##### This presentation is build in MDX Deck:
Great tool that converts MD(X) files to a presentation
[MDX Deck](https://github.com/jxnblk/mdx-deck)
